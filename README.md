## README de la viborita
Este es un proyecto de la viborita, un juego clásico donde el jugador controla una serpiente que se mueve por la pantalla, comiendo comida para crecer y evitando chocar contra las paredes o su propio cuerpo. El objetivo del juego es lograr la mayor puntuación posible antes de que la serpiente muera.
### Que hice en este proyecto:
- Implementé la lógica del juego utilizando C# y .NET 10.0.
- Diseñé una interfaz gráfica simple para mostrar la serpiente, la comida y el puntaje.
- Agregué controles para mover la serpiente en las cuatro direcciones (arriba, abajo, izquierda, derecha).
- Implementé la detección de colisiones para determinar cuándo la serpiente choca contra las paredes o su propio cuerpo.
- Agregué un sistema de puntuación que incrementa cada vez que la serpiente come comida.
### Como funciona el juego:
1. El jugador inicia el juego y la serpiente aparece en la pantalla.
2. El jugador utiliza las teclas de dirección para mover la serpiente.
3. La serpiente se mueve continuamente en la dirección seleccionada.
4. Cuando la serpiente come comida, crece y el puntaje aumenta.
5. El juego termina cuando la serpiente choca contra una pared o su propio cuerpo, mostrando el puntaje final.
